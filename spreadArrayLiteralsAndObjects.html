...spread Array Literals

We can also use the spread method with array literals.

The order in which we call the spread array is important, as that will be the way it will be ordered in the new array.

The original arrays are not changes, we are basically taking a copy of the array we spread for our new array.

We can also add items on top of the spread arrays.

And similarly, we can spread an array item into a new array, where every character of the item is a new item of a new array.

...spread Objects

We can copy, tweak, combine object properties, etc.

IF THERE IS A CONFLICT, LIKE A SAME KEY WITH A DIFFERENT VALUE FOR EACH OBJECT'S PROPERTIES, THE LAST ONE TO BE SPREAD WILL KEEP ITS VALUE!

In this case, both xx and yy have a color key, but only yellow is seen once both objects are combined into the new one, blue is just eliminated.

We can use this to retrieve user info from a form, and add our own info to a new object, like assigning a unique ID for each user for quick access to his/her information, or granting special permissions.

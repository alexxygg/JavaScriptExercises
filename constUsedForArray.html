const variable used for arrays

We can use a const for our array variable, while still adding, removing and modifying the content inside the array. IT WORKS:

const array = ["one","two"];

array.push("three");

array;

["one","two","three"];

What we can not do is modify the array as an element. We can not change the array to a number or a string, as it was set as an array in the first place.

We can not even change an array by updating it to the exact values, since JS would now see it as a different reference.

const array = ["one","two"];

array = ["one","two"];

ERROR!!!

That woud create 2 arrays, which is impossible since we set our array to be a constant variable, it can not be changed!

This is good because we can rely that our array container will be the same.

We can renovate inside the house, but not change all the walls and roof.

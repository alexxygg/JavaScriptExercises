String methods

BY entering a string in the console and adding a dot right after it, we can see all the different methods we can use.

SYNTAX --> thing.method();

.toUpperCase()

let msg = "leave me alone please."

msg.toUpperCase();

"LEAVE ME ALONE PLEASE."

Now we can store the new string in a new variable.

let angryMsg = msg.toUpperCase();

msg;

"leave me alone please."

angryMsg;

"LEAVE ME ALONE PLEASE."

These are the 2 variable we made.

toLowerCase()

Does the opposite of toUpperCase.

let angryLetters = "ABCDEFGH";

angryLetters.toLowerCase;

'abcdefgh'

Here, again, the resulting value is not saved, if we have any plans for it, we need to create a new variable.

let happyLetters = angryLetters.toLowerCase();

happyLetters;

'abcdefgh'

And we can still retrieve our original angryLetters.

angryLetters;

"ABCDEFGH"

trim()

This one creates a new string without the whitespaces at the beginning or end, no matter how many there are on either side. It keeps all the spaces in between.

let userInput = " hi my name is john "

userInput.trim();

'hi my name is john'

Chaining methods

We can chain methods to have multiple actions done for the end result.

let lazyTypedGreeting = " hello, my name is john ";

Here the string has spaces on both sides, and we want to make it all upper case too. SO we will chain two methods to accomplish this.

lazyTypedGreeting.trim().toUpperCase();

'HELLO, MY NAME IS JOHN'

We only had to add the second method after the first! The spaces were trimmed and all letters were made uppercase.

const message = " TASTE THE RAINBOW! ";

let whisper = message.toLowerCase().trim();

whisper;

'taste the rainbow!'

null and undefined

Also primitive value types.

undefined

It results when a value is unknown or not defined yet.

let x;

Here, x is declared, but it does not have a value given, so it is undefined.

""Hello"[9];

Here, we are trying to retrieve a value at index 9, but the last index in this string is 4. So the result would again be undefined.

null

It is a concept for saying nothing. Just nothingness.

We can set the value of our variables to null if wanted.

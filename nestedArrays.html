Nested arrays

We can have multile arrays in one array. They are some times called multi-dimensional arrays, or nested arrays.

Here we made a 4 by 4 array set, so there are 4 arrays with 4 items on each.

We need commas between each array inside our main array.

We can access an array by using the square brackets. If we only use one index, we would access the array at that index.

To access an item from one of the arrays, we have to add a second set of squared braces with the index of that specific item.

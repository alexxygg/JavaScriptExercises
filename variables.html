Variables

They help us resuse our code.

It is like giving your value a label, we store a name for it to come back to it when needed.

let someName = value;

let year = 1985;

let myLuckyNumber = 37;

let octopusLimbs = 8;

Retrieving varlues

let numShoePairs = 20

let numSocks = 50

let feetThings = numShoePairs + numSocks

If we multiple numShoePairs by 5, the result would be 100. But if we check the value of numShoePairs afterwards, the value would still be 20.

Here we can play with changing the values of any of the variables we made, but the result would not be updated. numSocks + 5 would result in 55, but checking numSocks after would only show 50.

Updating variable values

We can change numShoePairs to 40 in various ways.

We can declare numShoePairs = 40;

We can declare numShoePairs = numShoePairs + 20;

Checking the value of numShoePairs after doing one of the above would give us 40 now!

Checking the value of feetThings would not give us an updated value of numShoePairs + numSocks, it would return the value we stored at the time it was declared only. So although numShoePairs is 40 now, and numSocks is 50, we would not get 90 back, we would get 70. The value would not update unless done manually. We can simply declare the variable again: let feetThings = numShoePairs + numSocks; and now feetThings would result in 90!

Shorthands for updating a value.

We can also increment, reduce, multiple and divide with the shorthands:

variable += 1;

let numLives = 9;

numLives += 5

Now numLives has a value of 14!

let numLives = 9;

numLives -= 5

Now numLives has a value of 4!

let numLives = 9;

numLives *= 5

Now numLives has a value of 45!

let numLives = 9;

numLives /= 5

Now numLives has a value of 1.8!

let bankBalance = 100;

bankBalance /= 2;

bankBalance += 10;

Now bankBalance has a value of 60!

variable ++ and variable --

This shorthand is useful for changing the value by 1.

let score = 20;

score++;

score now has a value of 21!

let coins = 35;

coins --;

coins now has a value of 34!

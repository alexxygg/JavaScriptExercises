Promises in JavaScript

A promise is an object representing the eventual completion or failure of an asynchronous operation.

If we eat our vegetables, we may be promised to have ice cream. Like a guarantee of something happening depending on other factors.

The object can be pending, resolved or rejected. The status changes.

We wait for the retuned value to run code 1 or code 2, depending on the result.

Arrays

They are an ORDERED collection of values.

We need square brackets, like when using indexOf.

Here is an empty array:

let numbers =[];

It is an object.

Here are two with values.

ler lottoNumbers = [7, 25, 9, 12, 2, 16];

let products = ["iPhone", "iPad", "iWatch"]

Since they are ordered, we would have iWatch at index 2.

Here we have the days of the week in an array called "days".

We can check on our Dev Tools to see the content of the array including at which index each item is, and also to see its lenght(The number of items it has).

We can also see the lenght with:

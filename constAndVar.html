const and var

These are 2 other ways to declare a variable, other than the most commonly used let.

const

It helps us declare a constant variable.

If we use const to declare a variable, we can not change or update the value!

const luckyNumber = 7;

It can not be changed anymore, only on the line in which it is declared.

We can store things we are sure will not change, like tools.

const daysInAWeek = 7;

const pi = 3.14159;

const boilingPointC = 100;

const boilingPointF = 212;

var

The old way of creating variables, it used to be the only way to do so.

It still works, but not recommended.

It works like let.

Adjacent and Direct child selectors

These two help us be specific about the elements we are trying to style.

Adjacent selector

The adjacent selector uses a plus sign, which indicates we want the element that goes right after the first one.

Below we have 3 buttons.

We only want to style the buttons that have a paragraph right before it.

We used this code to achieve this: p + button { background-color: tomato; }

Direct child selector

The direct child selector is used when trying to access an element focused more on the structure of your code.

Below we have many buttons, they are all inside a form, but we also have one outside the form.

We can access it with the selector we just mentioned.

The code we used to reach it is: body > button { background-color: aqua; }

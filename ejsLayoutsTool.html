EJS Layouts Tool

We need to install it with npm i ejs-mate.

We will need to require it in our app.js file, we normally call it engine, since we use .engine(), but it can be called any other way.

  • const ejsMate = require("ejs-mate");
  • app.engine("ejs", ejsMate);

We are telling ejs to use this engine instead of the default one it uses to parse and process our code.

Now we can create a folder called layouts, inside our views folder, and start creating them.

Equality Operators

double equals ==

Checks if two values are equal, but does not check if types are also equal.

1==1;

true

Here there is no need to convert, as they are the same.

1=="1"

Here they are first converted to the same type and then compared. This is only done when possible.

Weird cases

null == undefined;

true

0 == false;

true

0 == ""; (empty string)

true

not equal !=

Checks if two numbers are not equal, again without checking for types.

1 != 2;

true (YES, They are NOT equal)

1 != "1";

false (NO, They ARE equal)

tripple equals === (strict equality)

This one does care about type, it does not do any attempt to change them first.

1 === "1";

false (NO, They are NOT equal)

0 === false;

false (NO, They are NOT equal)

triple not equal !== (strict inequality)

This one again checks for inequality for types, no attempt to compare different types.

1 !== "1";

true (YES, They are NOT equal)

2 !== 2

false (NO, They ARE equal)

Form events & preventDefault

When we have a form and the user submits it, the action we give the form element is where the form data will be returned to, or sent to by the user. Also, the website will change to that address by DEFAULT.

If we wanted to stay on the same webpage, we would use preventDefault

preventDefault()

We start by creating an event listener to the form when it is submitted.

Now we can use preventDefault() to change the default behavior of the form, which is to change the website to the form action.

To do this, we add preventDefault after the parameter name inside our function. This parameter is the auto-generated event object which we can name in any way when creating the function.

Now, we will stay on the website even after submitting.

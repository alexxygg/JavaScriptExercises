Changing Styles

We can access the CSS styles we've applied to any of our elements.

IN ORDER TO ACCESS A SPECIFIC STYLE, WE MUST USE CAMEL-CASE INSTEAD OF THE REGULAR STYLE NAMES SEPARATED BY A DASH.

text-decoration = textDecoration

background-color = backgroundColor

etc.

The style collection we are returned will not show us its value if linked from an external CSS document.

It only shows inline CSS Styles.

We can still make changes to the styles from the DOM, we simply can't see them from here.

We always use strings to enter our desired values.

Since we can't access the style values unless we declare them with inline CSS, there is an alternative.

After we've saved the collection or single element,(with id,tag name or class), we can use window.getComputedStyle("").style.

This problem starts because we sometimes have multiple CSS documents for a same webpage, and there are many CSS rules and hierarchies that change what the style actually applied ends up being.

WIth this command, we can get the last style that is applied once all the math has been done.

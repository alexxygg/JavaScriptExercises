Math.floor() ------FLOOR

It helps us remove the decimals after a number, NOT ROUND TO NEAREST INTEGER.

Math.floor(57.58999);

Math.ceil() ------CEILING

It helps us round up to the nearest whole number.

Math.ceil(65.1377);

RESULTS IN --> 66

Between specific numbers only.

Say we wanted a random number between 30 and 35.

First we need to count the number of options, which in this case would be 6: (30, 31, 32, 33, 34, 35).

So we can now place 6 to our *.

We also need to change the + to 30, since it is the first number we want to include.

Math.floor(Math.random() *6) + 30; -->Between 30 and 35!

Math.floor(Math.random() *6) + 30; -->Between 54 and 61!

Others

The Mtah object helps us do other things we can normally do without it.

Like getting the result of a number to the power of another number.

Math.pow(7, 3);

This is 7 to the power of 3, or 7 * 7 * 7.

Which we know we can get with: 7 ** 3;

Both result in 343.

Dice!

If we wanted to make a dice game, we can use the information we learned here to achieve it!

FIrst we will need to generate a random number between 1 and 6.

IN this case we can make them const variables, since the algorhitm used will not change.

Here we will have 2 dice, and we will create a message to let the user know what each number was, and also a sum of them.

const die1 = Math.floor(Math.random()*6) +1;

const die2 = Math.floor(Math.random()*6) +1;

rollMessage = `You rolled a ${die1} and a ${die2}. They sum to ${die1 + die2}!`

die1;

Strings

This is another primitive value type. We store text, symbols and numbers in them.

We can use single quotes or double quotes, but we can not start a string with one and finish with the other, we would get an error.

let username = "danny";

let username = "George";

let name = 'Alex';

They can have spaces once inside the quotes.

"My name is Alex."

If we are to include one or the other inside our string, we should do it logically.

"The TV screen is 55' diagonally."

'and I quote, "I will never do that again".'

let bestColor = "purple";

let quote = 'You had me at "hello"';

We should use the same type of string in a single document.

Creating a string with a number inside it does not mean it is a number.

We can make a string with a single character or none at all, called an empty string "".

Strings are indexed

Each character has a corresponding index, which is in relation to its position.

let animal = "Giant Octopus";

animal[0] = "G"

animal[1] = "i"

animal[2] = "a"

animal[3] = "n"

animal[4] = "t"

animal[5] = ""

animal[6] = "O"

animal[7] = "c"

animal[8] = "t"

animal[9] = "o"

animal[10] = "p"

animal[11] = "u"

animal[12] = "s"

"Giant Octopus".lenght;

Result: 13

"animal".lenght;

Result is 6, although the last letter, "l", is one less than that, index 5.

Spacing strings / Concatination

We can not change a string simply by updating its value, we would be creating a totally new string instead.

Here we try to concatinate these two strings, but since neither have a space at the end or start respectively, they are just next to each other in the result.

let firstName = "Alejandro";

let lastName = "Perez";

firstName + lastName;

"AlejandroPerez"

Here we create a string that only has a space in it, and add it in between our two original strings.

firstName + " " + lastName;

"Alejandro Perez"

We can store this new string in a new variable like this:

let fullName = firstName + " " + lastName;

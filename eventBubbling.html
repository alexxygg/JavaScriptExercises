Event Bubbling

When we have multiple events for various HTML elements nested withing one another, we can accidentally trigger other events we didn't want to.

Example:

If we have a div with a section inside, and that section has a button inside:

If each of these 3 elements have an onclick event, and we happen to click the button, which is the element most deeply nested in the example, we will trigger all 3 events!

We may sometimes want an event to bubble, like a wave or chain-reaction.

stopPropagation()

We can use this method on our event object to make sure the event doesn't bubble up, or trigger other events which may be chained to the element.

Clicking the nested elements will no longer trigger the outer events!

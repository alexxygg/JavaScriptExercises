if() Conditional

The if conditional helps us make decisions.

LetÂ´s say we declare a variable:

let stars = 5;

We can now use the if conditional to make a decision.

Here it is:

if(stars === 5){

("You have 5 stars.");

}

In the console we would see:

You have 5 stars.

If the code above was false, anything inside the curly braces would not run.

We added two console.log message, one before and one after the if conditional statement, those will run regardless.

random example

let random = Math.random();

if (random > 0.5) {

console.log("If you can see this, your number was more than 0.5!");

console.log(random);

}

Succesful attempt:

If you can see this, your number was more than 0.5!

0.7659067413046483

We could add another if conditional for when the number is less than 0.5. But there are easier ways to do so...

Comparison Operators

Greater than >

Helps us determine if a number on the left of it is greater than the other number.

7 > 1;

true

254 > 65;

true

25 > 200;

false

Greater than or equal to >=

Helps us determine if a number on the left of it is greater than or equal to the other number.

2 >= 1;

true

254 >= 254;

true

25 >= 26;

false

< Less than

Helps us determine if a number on the right of it is greater than the first number.

74 < 700;

true

25 < 81;

true

100 < 25;

false

<= Less than or equal to

Helps us determine if a number on the right of it is greater than the first number.

74 <= 73;

true

25 <= 24;

false

100 <= 100;

true

Decimals and Negative Numbers

They work the same as integers.

-5 > 2;

false

2.3 > 2;

true

-5<5;

true

5.5<=5.5;

true

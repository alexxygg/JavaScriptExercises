Infinite Loops

It can cause the computerÂ´s browser to crash, it uses all the memory that JS offers because the loop never stops.

An example is a for loops that will never reach its end:

for(let i = 10; i >= 0; i++)

The code will keep on running as long as the variable is more or equal to 0.

The variable already starts at more than 0, 10. So every time the code runs it will add one more, and it will always be true that our variable is more than 0.

The code will crash the browser and force quit or close.

We can fix them by thinking logically, and paying attention to our code.

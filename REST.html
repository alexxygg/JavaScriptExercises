REST

Representational State Transfer

It is a set of guidelines, ideas and principles for how clients and servers communicate.

If we want a CRUD application, (Create, Read, Update, Delete), we must have ways of doing all this within the application.

A RESTful website like Twitter has all the CRUD options.

They have a uniform interface, a consistent URL pattern matches with HTTP verbs.

We can send a post, get, update or delete HTTP request to Create, Read, Update and Delete!

The DOM

The DOM or Document Object Model is a JavaScript representation of a webpage.

It's a window into the contents of said webpage.

It is the sum of objects that we can interact with via JS.

HTML and CSS go in, and a JS object comes out.

There is a tree that shows us the relationship of each element with each-other.

They are nested respective to the parents or children, etc.

The document object includes everything in the webpage. It is our entry to the DOM. We can see representations of all our content, it is the root of our tree.

We can see it with console.dir(document).

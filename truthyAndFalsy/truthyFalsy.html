Truth and Falsy values

We have true and false, but we also have truthy and falsy values.

They are an inheritance they naturally have from JS.

ALL JS values are truthy... with the exception of:

  • false
  • 0
  • " " (empty string)
  • null
  • undefined
  • NaN

this Keyword in Methods

It allows us to access other properties on the same object.

It is commonly used in methods from objects.

We can reference properties by using this, since JS automatically assumes that we are referring to the object where we are using the word in.

The value of 'this' depends on where it is invoked, context is important regarding the function

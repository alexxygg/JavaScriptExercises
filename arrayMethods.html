Array Methods

There are many built-in array methods. Like the ones we have used with strings.

They need parenthesis just like others we´ve seen.(indexOf(),toUpperCase() etc.)

IT IS IMPORTANT to notice that when we use a method on a string, the value we are returned is not the same string, but updated. The original string will still have the same values exactly as before, we only get a returned string to be used if we need, or to be stores in a NEW variable.

With array methods, we ARE changing the original item!!!

.push() and .pop()

They work like a stack, we are adding and removing items from the END of the array.

.push()

Helps add an item to end of array

We can add multiple items at the same time, just separated by a comma.

.pop()

Helps remove an item from the end of the array.

This method doesn´t need an argument or value, since it will simply remove the last existing array item.

We can save the removed value in a variable.

Our current array names2 has "Howard" at the end.

Here we are removing him and saving him on a new variable at the same time.

.shift() and .unshift()

We are adding and removing items from the BEGINNING of the array.

.shift()

Helps remove an item from the beginning of the array.

We can use it without entering a value or argument, since we are just removing the first array item.

.unshift()

Helps add an item to the beginning of an array

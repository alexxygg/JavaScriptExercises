Functions

Functions allow us to reuse code. They are used very often

First we define the function, later we can run or execute it.

The code won't be ran until we explicitly do so.

WE SHOULD AVOID CALLING A FUNCTION BEFORE WE DEFINE THEM

To execute a function we need to type the function name followed by opening and closing parenthesis.

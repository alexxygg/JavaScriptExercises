return Keyword

The return keyword helps us make use of whatever value we obtain after our code runs, any output.

When we use built-in methods, we get values retuned.

Like when we use .toUpperCase() and get a string back in uppercase version.

Or when we use .indexOf() and get an index number for the array back matching the character or value we were looking for.

We can save those returned values in variables by defining them.

The return keyword stops the code from running, when the return keywords is used, the code is paused.

For example, if we use return after an if statement, there's no need for an else statement, the code will simply stop running if that conditional is not met.

We can only return one thing at the time, it can be one string, one array, one object, but only one thing.

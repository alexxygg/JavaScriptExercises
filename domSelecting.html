Selecting

First we select, so we can later manipulate.

getElementById("")

We can select a specific JS Object with getElementById.

We can then save the object to a variable, to later access the object along with all the object properties in a single place.

Here we access an element with an id called toc, short for table of contents.

We then save the JS object to a variable called toc. Finally, we access the object properties with console.dir() and the variable between the parenthesis.

If we had a heading and an image with ids of mainheading and unicorn:

const heading = document.getElementById("mainheading")

const image = document.getElementById("unicorn")

getElementsByTagName("")

Here we get ALL elementS by tag name, not just one element by Id.

This will return an HTML collection, which is all the elements that match the tag we specify, in a single JS object.

Here is an example, we have 2 images in a webpage, we are accessing the second one by specifying the index when we console.dir.

We can iterate over the array of objects we are returned.

IT WILL NOT WORK LIKE AN ARRAY, SINCE THEY ARE BOTH JS OBJECTS. WE ARE GETTING AN HTML COLLECTION, CONSISTING OF VARIOUS OBJECTS.

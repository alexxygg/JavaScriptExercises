Naming Variables

The name is an identifier, and it should be descriptive.

They can have numbers:

let user254 = 5;

But it can not be the first character, it will give us an error.

Best Practices

undercase

We should not start with an undercase: let _hey = true;

CamelCase

It is possible to name a variable nameofvariable, but we should uppercase each first letter of words after the first one.

oneTwoThreeFour

theNameOfMyVariable

under_case and under-case is bad practice, we should use underCase.

Clearness

gameOver is a bad name, it would be more clear if we call it isGameOver.
